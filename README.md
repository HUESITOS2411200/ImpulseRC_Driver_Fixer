# ImpulseRC Driver Fixer

<p align="center">
  <img src="logo.png" />
</p>

This application will attempt to install the correct serial port and DFU drivers for using STM32 F3, F4 and F7 based flight controllers on Windows. If you have trouble running the Driver Fixer you may need to install the [Microsoft .NET Framework v4.5](https://www.microsoft.com/en-au/download/details.aspx?id=30653)
